Class: GClabel

GClabel

creates a Raphael ellipse or rect

<private> new GClabel

Creates a Raphael ellipse or rect and contains some information about that element. This represents a label in either the behavioral view or the communication view.
Parameters:
Name Type Description
x int The x ordinate of the top left corner of this element.
y int The y ordinate of the top left corner of this element.
text String The text of the label.
shape String The shape of the label. Possible values are "rectangle", "roundedrectangle", "roundedrectanglemulti", "circle", "ellipse" (default: "roundedrectangle")
id String The id of the label.
belongsToPath boolean This indicates whether the label belongs to a path.
