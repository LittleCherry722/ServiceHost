text :String

The label of this node. When the type of this node is either send or receive, this value will not be displayed. Instead the letters "R" (receive) or "S" (send) will be displayed in the graph. When this is an end node, nothing will be displayed. In most cases this attribute will hold a short action description.
Source:
  • node.class.js, line 116

type :String

This is the node's type. It can be set to "action", "send", "receive" or to "end". It defaults to action. When the type is set to either "send" or "receive" it will be displayed as a circle with the letter "S" or "R" in it on the graph. An action node will be displayed as a rectangle holding the text stored in the node's text attribute.
