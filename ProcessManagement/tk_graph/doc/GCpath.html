Class: GCpath

GCpath

creates a Raphael path

<private> new GCpath

Creates a Raphael path and contains some information about that path. This represents an edge in the behavioral view.
Parameters:
Name Type Description
startx int The x ordinate of the starting point.
starty int The y ordinate of the starting point.
endx int The x ordinate of the ending point.
endy int The y ordinate of the ending point.
shape String The shape of the path. Possible values are "I", "L", "Z", "U", "S", "C", "G", "SI", "ZU", "UI"
text String The text to display on the edge's label
id String The id of the edge.
